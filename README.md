<h2>LandmarkDetail</h2>

This iOS application displays detailed information about landmarks using SwiftUI. The main component, LandmarkDetail, showcases a landmark's map location, image, name, park, state, and description. This project serves as an example of how to structure a SwiftUI view to present detailed content about a specific item.
